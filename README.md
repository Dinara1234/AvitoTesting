# Автотесты для API Avito

Проект содержит набор автотестов на Java + Rest Assured для проверки API микросервиса объявлений (тестовое окружение `https://qa-internship.avito.com`).  
Тесты покрывают основные эндпоинты: создание объявления, получение по ID, получение по продавцу, получение статистики (v1 и v2).

## Требования

- **Java 11 или 17** (рекомендуется 17)
- **Maven 3.6+**


## Установка и запуск

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/Dinara1234/AvitoTesting.git
   cd AvitoTesting

2. **Собрать проект и скачать зависимости**

   ```bash
   mvn clean compile

3. **Запустить все тесты**


   ```bash
   mvn test

После выполнения отчёт появится в консоли.
